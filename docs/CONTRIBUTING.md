# Contributing

Thank you for wanting to contribute! Suggested workflow:

1. Fork the repository and create a feature branch.
2. Run tests (none included currently) and verify the app runs locally:

```powershell
python -m venv venv
.\\venv\\Scripts\\Activate.ps1
pip install -r requirements.txt
python app.py
```

3. Make small, focused commits and open a pull request describing the change.

Code style
- Follow existing style in the repo. Keep changes minimal and well-documented.

Reporting bugs
- Open an issue with steps to reproduce, Python version, and OS details.
